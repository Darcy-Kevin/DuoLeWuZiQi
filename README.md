# 多乐五子棋应用UI自动化测试项目

## 项目简介
这是一个使用uiautomator2和pytest框架实现的多乐五子棋应用UI自动化测试项目。

## 目录结构
```
DuoLeWuZiQi/
├── src/                    # 源代码目录
│   ├── tests/              # 测试脚本目录
│   ├── config/             # 配置文件目录
│   ├── resources/          # 资源文件目录
│   └── utils/              # 工具函数目录
├── requirements.txt        # 项目依赖列表
├── setup.sh               # 环境设置脚本
└── run_tests.sh           # 测试运行脚本
```

## 快速开始
```bash
# 1. 设置环境
chmod +x *.sh
./setup.sh

# 2. 运行测试
./run_tests.sh
```
